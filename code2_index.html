Site web pour un restaurant

J'ai réalisé ce projet web pour un client travaillant dans le domaine de la restauration.

L'objectif? Permettre à ce restaurant de gagner en visibilitié et de proposer son menu de pizzas aux utilisateurs

Mon client, étant novice dans les nouvelles technologies, à la posssibilit de modifier facilement son menu en ligne grâce a une interface d'administration sur-même que j'ai réalisé.

Ce projet simple mais fonctionnel a donné entière satisfaction à mon client.

Technologies utilisées : HTML, CSS, DJANGO, PYTHON, HEROKU
